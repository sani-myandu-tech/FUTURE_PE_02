# FUTURE_PE_02
AI-generated UGC ad content for a Durban-based café using structured prompt engineering. Includes hooks, ad scripts, CTAs, and social media captions optimized for engagement and conversion.

TASK 2: UGC AD CONTENT (AI MARKETING)
 
This task focuses on creating high-converting UGC (User Generated Content)
ad scripts for a local business using structured prompt engineering.
 
 
BUSINESS CHOSEN
  Business  : Urban Bites Café
  Location  : Durban, South Africa
  Audience  : Students and young professionals
  USP       : Affordable gourmet meals with fast service
 
 
CONTENT CREATED
  * UGC video ad script
  * Multiple hook variations
  * CTA variations
  * Social media caption with hashtags
 
 
PROMPT ENGINEERING STRATEGY
A structured 3-step refinement process was applied, progressively
improving output quality with each iteration.
 
  Step 1 — Initial Generation
  Created a foundational UGC script using a structured prompt with
  a defined role, tone, target audience, and required format
  (hook, problem, solution, CTA).
 
  Step 2 — Improvement Prompt
  Enhanced engagement by layering in emotional appeal, stronger
  hooks, and Durban-specific local context to make the content
  feel authentic and relatable.
 
  Step 3 — Final Refinement
  Optimised for short-form video platforms (TikTok and Instagram
  Reels) with punchy sentence structure, improved pacing, and a
  specific action-driven CTA.
 
 
KEY IMPROVEMENTS
* Hooks optimised for the first 3 seconds — scroll-stopping
    and platform-native
  * Natural, relatable UGC tone — not scripted or overly promotional
  * Clear problem → solution → CTA structure maintained throughout
  * Content ready for both paid advertising and organic campaigns
 
 
OUTCOME
  The final content is realistic, engaging, and production-ready.
  Each iteration demonstrated measurable improvement in tone,
  emotional relevance, and platform suitability — validating the
  structured prompt engineering approach as an effective method
  for AI-assisted marketing content creation.
